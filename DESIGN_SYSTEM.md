# DESIGN_SYSTEM — Stats Learning Hub

このサイトの視覚設計を支えるトークン定義。**場当たり的にCSSを触らず、必ずここで定義した CSS 変数（`style.css` の `:root`）を参照する。**

## 設計思想

- **目指す印象**: 落ち着いた学術的ミニマル。装飾でなく「読みやすさ・視線誘導・余白」でスタイリッシュさを出す。
- **グラフが主役**: インタラクティブグラフ（Canvas / Plotly）が唯一の彩度の高い要素。**UI クロム（ナビ・枠・注記ボックス・スライダー）は無彩色〜低彩度に徹し、グラフの系列色と競合しない。**
- 情報密度が高い学習サイトなので、色数を絞り、階層はコントラストと余白で作る。

## カラーパレット

### グラフ系列色（`app.js` の `Plot.colors` / `PlotlyTheme.colors`・**UIから侵さない予約色**）
| 用途 | 色 |
|---|---|
| 系列1 | `#4f6df5`（青） |
| 系列2 | `#e4572e`（朱） |
| 系列3 | `#2a9d8f`（teal） |
| 系列4 | `#f4a261`（sand） |
| 系列5 | `#9b5de5`（紫） |
| 系列6 | `#577590`（slate） |
| 補助線 gray | `#98a2b3` / ink `#1d2433` |

→ グラフはこの6色＋グレーで青・朱・teal・紫などの高彩度を占有する。**UIアクセントはこの色域を避け、低彩度のスレート／墨で構成する。**

### UI トークン（CSS 変数）
| 変数 | 値 | 用途 |
|---|---|---|
| `--bg` | `#f6f6f7` | ページ背景 |
| `--panel` | `#ffffff` | カード・デモ・表の面 |
| `--surface` | `#eef0f3` | 低彩度の補助面（注記・まとめ・値バッジ） |
| `--ink` | `#101014` | 構造色（ヘッダー/ヒーロー/表頭/ボタン）・本文 |
| `--ink-2` `--ink-3` | `#1c1c22` `#2a2a33` | 墨の階調 |
| `--muted` | `#6b7080` | 補足テキスト |
| `--muted-2` | `#8a8e9c` | ラベル・カウンタ等の最弱テキスト |
| `--nav-ink` | `#43485a` | サイドバーのリンク文字 |
| `--line` | `#e5e6eb` | 罫線（弱） |
| `--line-strong` | `#d5d6dd` | 罫線（強）・入力枠 |
| `--hover` | `#ececef` | ホバー面 |
| **`--accent`** | **`#3b4557`** | **単一アクセント（スレート墨）**。選択状態・見出し縦バー・スライダー・リンク・フォーカス |
| `--accent-deep` | `#2a3242` | アクセントの濃色（値バッジ文字など） |
| `--ok` `--ok-soft` | `#0f766e` `#e8f5f2` | クイズ正解（グラフなし領域限定） |
| `--ng` `--ng-soft` | `#c2410c` `#fdf0e8` | クイズ不正解（同上） |

**アクセント選定理由**: 旧アクセント `#4f46e5`（インディゴ）はグラフ系列1 `#4f6df5`（青）とほぼ同一で、UIとグラフが同化していた。低彩度のスレート墨 `#3b4557` に変更し、グラフの青を唯一の「青い信号」として際立たせる。白背景・白抜き文字ともコントラスト比 ≈10:1 でAAクリア。

### ダークモード
**当面は light のみ（対応しない）と決定。** 理由: KaTeX は暗色テキスト前提、Plotly/Canvas の系列色は明色背景向けに最適化されており、ダーク化は「グラフのコントラスト・系列色を装飾目的で変えない」ガードに抵触するリスクが高い。将来対応する場合はグラフのテーマ切替を含む独立課題とする。

## タイポグラフィ

- **フォント**: `--font` = `"Inter","Noto Sans JP","Hiragino Sans","Yu Gothic UI",system-ui,sans-serif`。数式は KaTeX（Computer Modern）、コードは等幅フォールバック。日本語＋KaTeX 混在でベースラインが破綻しないよう本文 `line-height:1.85`。
- **サイズスケール（rem 基準ではなく px 実測、本文15px を基準に約1.2倍間隔）**:
  | 役割 | サイズ | 太さ |
  |---|---|---|
  | ヒーロー h1 | 34px | 900 |
  | トピック h1 | 28px | 900 |
  | セクション見出し h3 | 18px | 800 |
  | デモ/まとめ見出し h2 | 17.5px | 800 |
  | 本文 | 15px | 400 |
  | 補足・注記 | 13.5–14px | 400 |
  | ラベル・カウンタ | 11–12.5px | 500–700 |
- **数式**: display 数式 `.katex-display` は `overflow-x:auto` で横スクロール可（本文幅を割らない）。インライン数式は行の中で自然に流す。

## 余白とレイアウト

- **余白スケール**: 8の倍数を基本（6/8/12/16/20/24/32/40）。
- **最大幅**: レイアウト全体 `max-width:1240px`。**本文カラムは可読性のため実効measureを制御**（1行あたり全角約40字前後を目安に、`#main` 内の本文ブロックに読み幅上限を設ける）。
- **グラフ領域**: デモは `--panel` の枠で囲み、Canvas 400px / Plotly 440px 高。UI枠はグラフを囲む「額縁」に徹し、内部の描画色に干渉しない。

## コンポーネント

| コンポーネント | スタイル規約 |
|---|---|
| **ヘッダー(topbar)** | 墨(`--ink`)ソリッド・sticky。サイトの identity アンカー |
| **サイドバー** | 白背景・`--nav-ink` 文字。選択は `--accent` 面＋白文字。グループ見出しは `--muted-2` の小キャップ |
| **カード(card)** | 白面・`--line` 枠・角丸`--radius`。ホバーで浮き＋`--accent`の左バー |
| **注記(note)** | `--surface` 面＋`--accent` 左バー。**旧インディゴ面を廃し無彩色に** |
| **まとめ(keypoints)** | `--surface` 面・`--line` 枠 |
| **例題(example)** | 琥珀系（`--warn-*`）。グラフを含まない worked problem 専用 |
| **デモ枠(demo)** | 白面・`--line-strong` 枠。内部にコントロール＋グラフ |
| **コントロール** | スライダーつまみ/選択枠/値バッジは `--accent`・`--surface`。ボタンは墨ソリッド |
| **クイズ** | 選択肢は白＋`--line-strong`枠、正誤は `--ok`/`--ng`。番号バッジは `--accent` |
| **表(table.simple)** | 表頭は墨(`--ink`)白抜き・角丸。行罫線は `--line` |
