# 統計学習ハブ（Stats Learning Hub）

統計検定2級の知識を土台に、**準1級レベルの統計理論・実験計画法・機械学習・多変量解析**を、パラメータを操作できるインタラクティブなグラフで学ぶための教材サイトです。

🔗 **公開サイト**: https://sgt9863.github.io/Stats-Learning-Hub/

## 特徴

- **全21トピック**、すべてに触って動かせるグラフ付き
- 統計検定2級 → 準1級のギャップを埋める構成
- スライダーやボタンで「式の意味」を体感できる
- **ビルド不要・依存ゼロ**のバニラJS。数式表示に KaTeX（CDN）のみ使用

## ローカルで動かす

依存パッケージのインストールは不要です。任意の静的サーバーで配信してください（数式・グラフの描画にネット接続＝CDNアクセスが必要です）。

```bash
python3 -m http.server 5180 --directory .
# → http://localhost:5180/ を開く
```

## 構成

```
index.html      … ページの骨組み（ハッシュルーティングのSPA）
style.css       … スタイル
app.js          … コアランタイム（統計関数 Stats・描画ヘルパー Plot・ルーティング・render）
topics/
  prep1.js      … 準1級 統計理論
  doe.js        … 実験計画法
  ml.js         … 機械学習
  mva.js        … 多変量解析
```

各トピックは `window.STATS_TOPICS` に `{ section, id, title, summary, body, demo }` を push する形で定義します。`demo.draw(canvas, params)` が Canvas にグラフを描き、`demo.controls` のスライダー／ボタン操作で再描画されます。DOM への書き込みは原則 `render()` に集約し、デモ操作時のみ Canvas を再描画します。
