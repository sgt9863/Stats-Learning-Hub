# 統計学習ハブ（Stats Learning Hub）

高校数学（数III）の知識を土台に、**統計検定準1級の統計理論・実験計画法（応用）・ケモメトリクス**を、図とインタラクティブなグラフで学ぶ教材サイトです。対象は統計検定2級レベル。

🔗 **公開サイト**: https://sgt9863.github.io/Stats-Learning-Hub/

## 特徴

- **全42トピック**。多くは触って動かせるグラフ、要所は **Plotly** の 3D で回して確かめられる
- **数学の基礎（数III）** から出発し、線形代数・微分の直感を作ってから統計へ
- 統計検定**準1級の出題範囲**を体系的にカバー（[公式範囲表](https://www.toukei-kentei.jp/)に対応）
- **ビルド不要・依存ゼロ**のバニラ JS。数式表示に KaTeX、グラフの一部に Plotly（いずれも CDN）

## セクションの分け方

カテゴリが重なる項目は**準1級を優先**して 1 か所にまとめ、他セクションには**準1級範囲外の発展内容だけ**を置いています。

| セクション | 主なトピック |
|---|---|
| 📐 数学の基礎 | ベクトルと行列 / 行列の積・逆行列 / 固有値・固有ベクトル / 分散共分散行列(3D) / 偏微分・勾配で最小化(3D) / 変数変換と線形化 |
| 📘 準1級 統計理論 | 確率分布・中心極限定理・最尤推定・仮説検定・信頼区間・ベイズ・デルタ法・ブートストラップ / 単回帰・重回帰と計画行列(3D)・多重共線性・ロジスティック回帰・過学習・モデル選択(AIC) / 分割表・カイ二乗独立性・ノンパラ検定 / 相関・主成分分析・判別分析・クラスター分析・因子分析・k-means / 分散分析・交互作用・直交表 / 時系列の基礎 |

## ローカルで動かす

依存パッケージのインストールは不要です。任意の静的サーバーで配信してください（数式・グラフの描画にネット接続＝CDNアクセスが必要です）。

```bash
python3 -m http.server 5180 --directory .
# → http://localhost:5180/ を開く
```

## 構成

```
index.html      … ページの骨組み（ハッシュルーティングのSPA）
style.css       … スタイル
app.js          … コアランタイム（統計・線形代数の Stats / Canvas描画の Plot / Plotly共通テーマ / ルーティング / render）
topics/
  math.js       … 数学の基礎
  prep1.js      … 準1級 統計理論（確率・推定・検定・ベイズ）
  prep1b.js     … 準1級 統計理論（重回帰・多重共線性・モデル選択・分割表・ノンパラ・ブートストラップ・時系列）
  ml.js         … 準1級に再配置（単回帰・過学習・ロジスティック・k-means・交差検証）
  mva.js        … 準1級に再配置（相関・PCA・LDA・クラスター・因子分析）
  doe.js        … 実験計画法（基礎は準1級に、応用は実験計画法セクションに）
  chemo.js      … ケモメトリクス（PLS / PLS-DA / OPLS-DA）
```

各トピックは `window.STATS_TOPICS` に `{ section, group, id, title, summary, body, demo }` を push して定義します。デモは 2 種類：

- **Canvas デモ** … `demo.draw(canvas, params)` が `Plot` ヘルパーで 2D グラフを描く
- **Plotly デモ** … `demo.plot(div, params, Plotly)` が 3D 曲面・等高線・スコアプロットを描く（`PlotlyTheme` で共通テーマ）

`demo.controls`（range / select / button）の操作で再描画されます。DOM への書き込みは原則 `render()` に集約し、デモ操作時のみグラフを再描画します。
